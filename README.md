# Rimuru LP — клиент дежурного (Go)

VK long-poll юзербот-клиент для системы дежурных [rimuruproject.ru](https://rimuruproject.ru).
Запускается на вашем аккаунте, ловит ваши команды в беседах (префиксы `.л` / `!л`) и
пересылает сигналы на хаб. Один процесс = один VK-аккаунт.

## Установка

1. Скачайте бинарь для вашей ОС со страницы [**Releases**](../../releases):
   - **Windows** — `rimuru-lp-client_windows_amd64.zip`
   - **macOS** — `rimuru-lp-client_darwin_arm64.tar.gz` (Apple Silicon) или `..._amd64...` (Intel)
   - **Linux (Ubuntu/Debian)** — `rimuru-lp-client_linux_amd64.tar.gz` (или `arm64`)
2. Распакуйте, положите бинарь в отдельную папку.

## Первый запуск

```bash
# Linux/macOS
chmod +x rimuru-lp-client
./rimuru-lp-client

# Windows — просто запустите rimuru-lp-client.exe
```

При первом запуске программа попросит **VK-токен**. Вставьте ссылку с токеном
(с `access_token=...` в адресе) или сам токен. Он сохранится в `config.json` рядом с
программой — **только на вашем компьютере**.

## ⚠️ Безопасность

- **Ваш `config.json` содержит ваш VK-токен.** Никому не показывайте этот файл и не
  заливайте его никуда. Файл создаётся с правами `0600` (доступ только вам) и добавлен
  в `.gitignore`.
- Программа **не содержит** и **не запрашивает** никаких чужих ключей — только ваш токен.
- Токен вводится вами и хранится локально; на сервер уходят только сигналы команд.

## Поддерживаемые платформы

| ОС | Архитектуры |
|----|-------------|
| Windows | amd64 |
| macOS | amd64 (Intel), arm64 (Apple Silicon) |
| Linux (Ubuntu/Debian) | amd64, arm64 |

## Сборка из исходников

```bash
go build -o rimuru-lp-client .
```
Нужен Go 1.23+. Бинарь статический (`CGO_ENABLED=0`) — без внешних зависимостей.
